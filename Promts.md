# AI Usage Documentation

## Overview

AI assistance was used during the development of this Kanban Task Manager for UI improvements, project planning, code review, and documentation refinement.

The core implementation, component structure, state management, task movement logic, localStorage integration, and overall project assembly were completed manually.

---

## Prompt 1: Dashboard UI Improvements

Improve the dashboard overview section of a React Kanban Board by suggesting better layouts, spacing, colors, and card designs for task statistics.

### Purpose
Used to enhance the visual appearance of:
- Total Tasks
- In Progress Tasks
- Completed Tasks
